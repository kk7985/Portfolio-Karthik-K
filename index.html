Machine Learning
2025
Pythonscikit-learnPandasMatplotlibSeaborn
Overview

Used a Retail & Product Categorization Dataset. Clustering is applied to group customers based on their behavior, helping businesses identify patterns and drive actionable insights. Completed as my second project at Infotact Solutions.

Problem Statement

The dataset has a large customer base but lacks insights into behavior, value and purchasing patterns. Without segmentation, marketing and retention strategies are generic, leading to missed opportunities for targeted promotions and improved customer loyalty.

Project Objective

Clean and preprocess customer transaction data, analyze purchasing behavior using metrics like income, age and spending score, and apply clustering techniques to segment customers, enabling better insights through visualization of customer groups.

Steps in Process
Used a Retail & Product Categorization Dataset (200 records) containing demographics, purchase behavior and transactional details.
Performed EDA (univariate and bivariate) to understand data patterns, relationships and feature distributions.
Identified Annual Income vs Spending Score and Age vs Spending Score as strong features for clustering.
Applied the Elbow Method to find optimal clusters (k=5) and performed K-Means Clustering, achieving a Silhouette Score of 71%.
Segmented customers into five distinct groups and interpreted each cluster to derive meaningful business insights.

Analyzed sport retail data to understand purchasing behavior and engagement patterns.
🔬
Conducted EDA to examine distributions and relationships between key variables.
📐
Determined the optimal number of clusters (k=5) using the Elbow Method.
✅
Applied K-Means clustering and validated results using Silhouette Score (0.71) — Excellent Clustering.
💡
Profiled each customer segment to derive actionable business insights.
Outcome
The customer segmentation divided customers based on transaction behavior and engagement patterns, identifying high value, frequent and occasional groups. The insights helped guide targeted marketing, personalized offers and strategies for improving customer retention.

Overview

Used specific Facebook account data by generating an API access token and extracting post data. Applied NLP and Machine Learning to identify trends, sentiment and user behavior through keywords, hashtags and engagement metrics. This was my final year mini project.

Problem Statement

Platforms like Facebook generate large volumes of unstructured user data, making analysis difficult. This project focuses on collecting and processing such data to extract insights like trends, sentiment and user engagement patterns, enabling better decision making.

Project Objective

Develop a system for collecting and analyzing data from Facebook using the Graph API. Focus on keyword and hashtag trends, sentiment analysis, topic modeling and time-based analysis to uncover insights and support better decision making.

Steps in Process
Collected data from a specific Facebook account using the Graph API by generating an access token. Extracted post-level data (captions, hashtags, likes, comments, shares, timestamps) and stored it in CSV format.

Analyzed data using statistics and visualization to identify patterns, trends and relationships.
Analyzed keywords and hashtags in post captions by measuring frequency and tracking trends over time using regular expressions and collections.Counter.
Classified text as positive, negative or neutral to understand opinions in post comments using the TextBlob library.
Used Gensim LDA to discover hidden topics and group related content into meaningful themes.
Analyzed changes in posting activity across different time frames (daily, weekly or monthly) to identify patterns.
Presented results using graphs and charts to make insights clear and support decision making.

🧠

📁
Applied topic modeling (LDA) to uncover hidden themes and time series analysis to track posting patterns over daily, weekly and monthly intervals.
📉
Visualized results with charts and graphs to extract actionable insights for marketing, content strategy and decision making.

College Campaign Data Analysis and Dashboard (SQL/Tableau)

2026
SQLTableauExcel
Overview

Analyzed a College Marketing Campaign dataset with 764 records using SQL and Tableau. The goal was to evaluate campaign performance, understand user engagement and provide insights for improving college marketing strategies. Metrics such as conversion rate, cost, revenue and ROI were analyzed across campaigns.

Problem Statement

Marketing campaigns generate large amounts of data but without proper analysis it is difficult to identify effective campaigns or optimize ROI. This project focuses on analyzing and visualizing campaign data to deliver insights for improving engagement and performance.

Project Objective

Analyze campaign performance using key metrics like revenue generated, Return on Investment and conversion rate to provide data-driven recommendations to improve future marketing strategies.

Steps in Process
Imported the dataset into PostgreSQL for data storage and processing.
Performed data inspection and cleaning by handling missing values, duplicates and inconsistencies.
Connected the cleaned data to Tableau for visualization.
Created dashboards and calculated key metrics: ROI = (Revenue - Cost) / Cost x 100
Conversion Rate = Conversions / Total Clicks x 100
Generated insights and recommendations to improve future campaign performances.
What I Did
🗃
Analyzed and processed data using SQL.
📊
Built interactive dashboards for campaign performance in Tableau.
🏆
Placement campaigns delivered the highest ROI (24.4%), making them the most profitable.
📈
MBA and UG Admission campaigns showed strong conversion rates above 23%, indicating effective targeting and lead quality.
⚠
Hackathon campaigns underperformed with low ROI (21.1%), low conversion rate (11.82%) and low revenue, suggesting a need to optimize or reconsider investment.
Outcome
Evaluated campaign performance by analyzing ROI, revenue and conversion metrics. Delivered data-driven insights through Tableau dashboards to enhance strategy and maximize overall performance.
